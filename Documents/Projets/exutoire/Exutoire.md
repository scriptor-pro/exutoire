# Exutoire

## Phase 1

 L'utilisateur entre ses pensées, une à la fois pendant une durée libre ou configurable. Interface minimaliste : champ texte et bouton. Enter/Control joue le même rôle que le bouton : enregistrer

## Phase 2

L'utilisateur trie ses pensées : 

+ Action concrète

+ Idée créative

+ Inquiétude / émotion

+ Information à garder

## Phase 3 (après le MVP)

Les actions → vont dans un système (agenda, todo)

Les idées → vont dans une liste projets

Les inquiétudes → soit tu planifies un moment pour y réfléchir, soit tu les laisses écrites

Les informations → archive
